# FlappyBird-AI
Inteligência Artificial e Rede Neural projetada para jogar Flappy Bird. Gym e OpenAi. 

## Instalação
```
$ git clone https://github.com/marlonsousa8/FlappyBird-AI.git
$ pip3 install -r requirements.txt
$ python3 main.py
```
